# Chapter 2 - Introduction
 
### Project Setup - Windows
1) Open Run Window

    ```
    START + R
    ```

2) Go to Command Prompt

    ```
    cmd
    ```

3) Go to Desktop

    ```
    cd Desktop
    ```

4) Create a Project Folder

    ```
    mkdir web
    ```

5) Go to Project Folder

    ```
    cd web
    ```

6) Open VSCode

    ```
    code .
    ```

7) Create HTML File

    ```
    index.html
    ```
